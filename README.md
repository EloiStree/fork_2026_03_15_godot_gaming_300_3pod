# 3P0D EDUCATION > GODOT > LOWPOLY_CHARACTER_SHOWROOM
Project template for a small exercise in integrating assets into a given project.

<br>

## PRESENTATION
This is a small project created as part of a five-day exercise in which participants are required to produce and integrate assets within a specified artistic style (low-poly with UV unwrapping on gradient textures). The project includes several character examples, as well as the character’s 3D base model. As the project is part of a workshop, there is no specific documentation for its implementation; it should be fairly intuitive.

<br>

## CONTENT
- Image of the wireframe for the basic character model (for the concept phase).
- 3D model of the character, along with examples.
- Godot project, which already includes a few characters.
